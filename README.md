# HSK 7–9 Vocabulary Quiz

Static GitHub Pages quiz site generated from `HSK-3.0-Levels-7-9-Word-List.pdf`.

Extracted rows: 5636

Files:
- `index.html` — quiz interface
- `vocab.js` — JavaScript vocabulary data
- `vocab.json` — raw JSON vocabulary data
- `hsk79_vocab.csv` — spreadsheet-friendly data

To publish:
1. Upload these files to a GitHub repository.
2. Go to Settings → Pages.
3. Set the source to the main branch/root folder.
4. Open the GitHub Pages URL.
